Arrays


An array is a collection of variables of the same data type referenced by a common name. Arrays become useful is situations where many elements of the same data type need to be stored or processed. All arrays consist of contiguous memory locations. Arrays can have data items of fundamental data types like int or float, or even of user-defined data types like structures.
Arrays can be of one-dimensional or multi-dimensional. A one-dimensional array is the simplest form of arrays and here we only consider those.
Let’s see how arrays are declared.
Suppose we want to declare an array , having 5 elements all of which are of integer type, we do this by: 

    int main() 
    { 
        int a[5]={100,200,300,400,500}; 
        return 0; 
    }

The above code creates an array like this:


If we write a code like this: 

    int main() 
    { 
        int a[]={100,200,300,400,500};    //size not specified 
        return 0; 
    }

without specifying the size, array a is created to accommodate the given elements.


We can create an array using user-inputs as: 

    int main() 
    {  
        int size;  
        cin>>size;     // entering the size of the array  
        int a[size];              // array declaration  
        for(int i=0;i>a[i];     //entering the elements in the array  
        return 0; 
    }

Accessing an element


An element in an array can be accessed by indexing the array. Arrays are indexed from 0 to (n-1) where n is the size of the array. This means that the first element in the array is indexed 0 and so on, and the last element is indexed (n-1). In general, for the ith element, the index is (i-1). For example, 

    int main() 
    {  
        int a[5]={100,200,300,400,500};  
        cout<<a[3];     //to print the 4th element  
        return 0; 
    }

The above code prints:

400

Another way of accessing the array elements is by using the fact that the name of an array always points to the address of the first element. The following code illustrates this: 

    int main()
