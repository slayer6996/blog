Doubly linked list


Make sure you are familiar with singly linked list in order to understand doubly linked list.


A doubly linked list similar to singly linked list has nodes as its elements, but the difference is that in doubly linked list a node can access next as well as previous node.

A node has data, and two pointers next and prev, containig address of next and previous nodes respectively.


In a doubly linked list we have 4 main operations, insertion of node at head and tail and deletion of node from head and tail.


Insertion at head:
	1.initialise a temp node.
	2. make temp->next point to the node to which is head is pointing to
	3. If head is not null, i.e if it is pointing to a node, that nodes'
	prev should point to our new node.
	4. head should now point to temp node.

Insertion at tail:
	1.check if head is NULL, if that's the case then call insert_at_head()
	2.If head!=NULL, make a temp node. 
	3.Make ptr node and traverse the linked list till the last node.
	4.At the last node, make its next point to temp.
	5. Make the temp nodes' prev point to last node. 

Deletion at head:
	1. if head==NULL, linked list is empty.
	2. make a pointer temp to store value of head.
	3. make head point to second node by head=head->next.
	4. now the new node head is pointing should have its prev pointing to 
	NULL.
	5. delete the temp.

Deletion at tail: 
	1.if head==NULL, linked list is empty.
	2.make pointer temp point to head.
	3.traverse the linked list till the last node. Now temp is pointing 
	to last node.
	4. access previous node from temp and make its next point to NULL.
	5. delete temp.

Following is the implementation of doubly linked list in C++

    struct node
    { 
        int data;
        node *next;
        node *prev;
    
        node(int val)
        {
            data=val;
            next=NULL;
            prev=NULL;
        }
    };
    
    class ll
    {
    private:
        node *head;
    public:
        ll()
        {
            head=NULL;
        }
    
        void insert_at_head(int val)
        {
            node *temp=new node(val);
    
            temp->next=head;
            if(head!=NULL)
                head->prev=temp;
            head=temp;
        }
    
        void insert_at_tail(int val)
        {
            if(head==NULL)
                insert_at_head(val);
            else
            {
                node *temp=new node(val);
                node *ptr=head;
                while(ptr->next!=NULL)
                    ptr=ptr->next;
                ptr->next=temp;
                temp->prev=ptr;
            }
        }
    
        void delete_at_head()
        {
            if(head==NULL)
                cout<<"empty\n";
            else
            {
                node *temp=head;
                head=head->next;
                head->prev=NULL;
    
                delete temp;
            }
        }
    
        void delete_at_tail()
        {
            if(head==NULL)
                cout<<"empty\n";
            else
            {
                node *temp=head;
                while(temp->next!=NULL)
                    temp=temp->next;
                temp->prev->next=NULL;
    
                delete temp;
            }
        }
    
        void display()
        {
            if(head==NULL)
                cout<<"empty\n";
            else
            {
                node *ptr=head;
                while(ptr!=NULL)
                {
                    cout<<ptr->data<<" ";
                    ptr=ptr->next;
                }
                cout<<"\n";
            }
        }
    };
