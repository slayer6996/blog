Mergesort


Merge function:

1. Make the merge function to merge two sorted arrays which takes the array, low, high and mid pointer which are pointing to the corresponding indices.

2. Determine the sizes of two arrays and make two dummy arrays and insert all the elements of the original arrays into these two arrays.

3. Now traverse both the arrays and compare the elements and update the original array filling the elements in their right position.

4. If any of the dummy array still has elements left to be inserted in answer array then insert them into it.


Mergesort function:

1. Make the function mergesort() which does the sorting.

2. Check for the condition if left pointer is smaller than right and if that holds true proceed as follows.

3. Find the mid element by (l+r)/2;

4. Now call mergesort function recursively for the left and right part of the array.

5. Call the merge function to merge the two sorted arrays.



Follwing is the code for mergesort in C++:

void merge(int ar[], int l,int mid,int r)
{
    int n1=mid-l+1;
    int n2=r-mid;

    int a[n1], b[n2];
    for(int i=0;i<n1;i++)
        a[i]=ar[l+i];
    for(int i=0;i<n2;i++)
        b[i]=ar[mid+1+i];

    int i(0),j(0),k(l);

    while(i<n1 && j<n2)
    {
        if(a[i]<b[j])
        {
            ar[k]=a[i];
            i++;k++;
        }
        else
        {
            ar[k]=b[j];
            k++;j++;
        }
    }

    while(i<n1)
    {
        ar[k]=a[i];
        i++;k++;
    }
    while(j<n2)
    {
        ar[k]=b[j];
        j++;k++;
    }
}

void mergesort(int ar[], int l, int r)
{
    if(l<r)
    {
        int mid=(l+r)/2;
        mergesort(ar,l,mid);
        mergesort(ar,mid+1,r);
        merge(ar, l, mid, r);
    }
}
