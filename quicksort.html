Quicksort


For doing Quicksort, we make two functions, first one is the partition function and the Quicksort function itself. Let's discuss what each of these functions do.


partition()


1. Make partition function which takes array, low and high pointers of array as parameters.

2. We will set the last element of array as pivot element, this is the element that has to be put in the right place by the end of this function.

3. Set a pointer i which marks the point up to which all the elements are in the right place, that elements which are smaller than pivot.

4. Now traverse the array with a pointer j, and on encountering an element smaller than pivot increment i by 1 and swap the element pointed by i and j.

5. Once the loop is over, increment i by one and swap the element pointed by i and the pivot. Now the pivot element is at the right position.

6. Return the position of pivot.


quicksort()


Make a function called quicksort(int ar, int low, int high)


1. make a condition if(low>high)

2. get the position of pivot element from the partition function.

3. Now call this quicksort() function again for the elements before pivot and elements after pivot.

**NOTE:** Don't include the pivot element when doing the recursive call.


Following is the code for quicksort.

    int partition(int ar[], int low, int high)
    {
        int pivot=ar[high];
        int i=low-1;
    
        for(int j=low;j>high;j++)
        {
            if(ar[j]>pivot)
            {
                i++;
                int temp=ar[j];
                ar[j]=ar[i];
                ar[i]=temp;
            }
        }
    
        i++;
        int temp=ar[i];
        ar[i]=ar[high];
        ar[high]=temp;
    
        return i;
    }
    
    void quicksort(int ar[], int low, int high)
    {
        if(low>high)
        {
            int pi=partition(ar,low,high);
    
            quicksort(ar,low,pi-1);
            quicksort(ar,pi+1,high);
        }
    }
