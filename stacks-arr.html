Stacks


Pre Requisites:

  • Arrays
  • Linked list

Stack is a Datastructure which can store elements of similar data types. It is a LIFO(last in first out) Datastructure, it means that the last element pushed into a stack is the first one to come out.

We will have a look at how a stack is designed and implemented in C++.
The commonly implemented functions of a stack are- push, pop, isempty and display.

We insert an element into a stack using push function. Remember that the element can be added only to the top of the stack. The deletion of an element is possible only from the top of the stack, this is done by pop function. The isempty function has return type bool which returns true if the stack is empty.


The code for implementing stack using Arrays is given below, the code is pretty much self explanatory.

    //implementing stacks with arrays
class stackk
{
private:
    int top;
    int size;
public:
    int a[1000];     //max size for stack=1000

    stackk()
    {
        top=-1;     //initially stack has no element
    }

    void push(int n)
    {
        if(top>=(1000-1))
            cout<<"stack overflow\n";
        else
        {
            a[++top]=n;     //increament top and push n
        }
    }

    void pop()
    {
        if(top<0)
            cout<<"stack underflow\n";
        else
        {
            cout<<a[top--]<<" popped\n";        //first print a[top] and then decrement
        }
    }

    bool isempty()
    {
        if(top<0)
            return true;
        else
            return false;
    }

    void display()
    {
        if(top<0)
            cout<<"stack is empty\n";
        else
        {
            for(int i=top;i>=0;i--)
            {
                cout<<a[i]<<"\n";
            }
        }
    }
};
