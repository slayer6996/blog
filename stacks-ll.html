Stack implementation using linked list


Stack can be implemented using a linked list. Earlier we say how we can implement stack using arrays but there is a limitation while implementing stack with arrays, the size of the stack is fixed. To overcome this problem, we use linked list.


The idea behind the implementation is same as we discussed earlier.
The code for implementing stack with linked list in C++ is given below, the code is pretty much self explanatory. 

    //implementing stacks using linked list

    class Stack
    {
    private:
        int data;
        Stack *next;        //pointing to the nextt element
        Stack *top;         //points to the top element
    public:
        Stack()
        {
            top=NULL;       //initialising top with NULL
        }
    
        //the idea is to put the new element at the beginning of linked list
    
        void push(int n)
        {
            Stack *temp= new Stack;
            temp->data=n;
            temp->next=NULL;
            if(top==NULL)
            {
                top=temp;
            }
            else
            {
                temp->next=top;
                top=temp;
            }
        }
    
        //to pop element, we delete the first node of linked list
    
        void pop()
        {
            if(top==NULL)
                cout<<"stack undeflow\n";
            else
            {
                top=top->next;
            }
        }
    
        void display()
        {
            if(top==NULL)
                cout<<"stack is empty\n";
            else
            {
                Stack *ptr=top;
                while(ptr!=NULL)
                {
                    cout<<ptr->data<<" ";
                    ptr=ptr->next;
                }
                cout<<"\n";
            }
        }
    
        bool isempty()
        {
            if(top==NULL)
                return true;
            else
                return false;
        }
    };
